## Installation

```bash
yarn add vue-cocomaterial-image
```

## Usage

```js
import CocoMaterialImage from 'vue-cocomaterial-image'
```

```html
<template>
  <div id="app">
    <CocoMaterialImage :imageId="151" foreground="#000" background="#fff" />
  </div>
</template>
```
